Studio Matrix Session Booking manages online session or meeting by helping the admin to create session/meeting slots for a WooCommerce product.

== Description ==
**Studio Matrix Session Booking** allows the admin to choose a product along with the duration of a session form all the products which were created via wooCommerce. Adding time slots for different days and showing them in the frontend so that the clients can choose from the available dates and time.

Major features in Studio Matrix Session Booking includes:
* Timezone compatibility
* Auto-load of WooCommerce Product
* Integration of choosing date and time on frontend without short codes.
* View of all the appointment in a calendar

== Installation ==
Installing the plugin is easy. Just follow these steps:

1. From the dashboard of your site, navigate to Plugins --> Add New.
2. Select the Upload option and hit "Choose File."
3. When the popup appears select the stmsb-session-booking.zip file from your computer.
4. Follow the on-screen instructions and wait as the upload completes.
5. When it will be finished, activate the plugin.
6. Go to the "General Setting" under Session Mgmt menu and setup the timezone first before creating Session.
